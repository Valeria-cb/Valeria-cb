# | Valéria Bueno |

**Em transição de carreira - ARQ | TECH**

Sou formada em Arquitetura e Urbanismo, com experiência no desenvolvimento de projetos arquitetônicos e urbanos, atuando de forma analítica, estruturada e orientada à entrega de soluções com qualidade.
Ao longo da minha trajetória, desenvolvi competências como atenção aos detalhes, pensamento crítico, visão sistêmica, organização, comunicação com equipes multidisciplinares e resolução de problemas, habilidades que hoje aplico na minha transição de carreira para a área de Tecnologia.

Atualmente, estou aprofundando meus conhecimentos em Desenvolvimento Front-end, estudando lógica de programação, JavaScript, HTML, CSS, Git e GitHub, além de desenvolver projetos práticos para consolidar meu aprendizado. Paralelamente, também estudo fundamentos de Qualidade de Software (QA) e testes de software, buscando compreender todo o ciclo de desenvolvimento de uma aplicação.
